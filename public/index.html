Welcome to myFlix API

The myFlix API is a RESTful API that provides access to a movie database. It allows users to retrieve information about movies, directors, and genres, as well as manage user accounts and favorites.

The API is built using Node.js, Express, and MongoDB, and follows RESTful principles to provide a simple and intuitive interface for developers to interact with the movie database.

The API is designed to be easy to use and flexible, allowing developers to integrate it into their applications quickly and efficiently. It supports a variety of request methods, including GET, POST, PUT, and DELETE, and returns data in JSON format. You can see more details in the repository README.md or here.

This is used by both the myFlix Angular Frontend and the myFlix React Frontend.
